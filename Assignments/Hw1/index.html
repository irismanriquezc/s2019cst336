Introduction

Computers can store a large amount of data and process it. Formal data structures help a programmer to structure large amounts of data into conceptually manageable relationships mentally. Sometimes we use data structures to do more: to quickly search or sort data, for example. Other times, we use data structures to do less. For instance, the stack concept is a limited form of a more general structure of data. These limitations give us guarantees that make it easier for us to understand our programs. Data structures also provide algorithmic complexity guarantees. To write good software, it is essential to select a suitable data structure for a job.

Some data structure examples are:

  • Vectors
  • Linked Lists
  • Trees
  • Hash Maps
  • Stacks
  • Queues
  • Graphs
