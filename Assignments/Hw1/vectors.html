A vector is similar to an array, in a sense that a series of elements is stored in the same variable name. Vectors are dynamically sized, which is an obvious advantage, unlike arrays.

The basic data structure we use for vectors is a dynamically allocated array. This makes it very easy to implement many of the operations. For example, the indexing to the vector occurs by applying the same index to the array

The benefits of vectors are that it pre-allocates space for future elements, so extra space required beyond what's necessary for the elements themselves, each element only requires the space for the element type itself (no extra pointers), and can re-allocate memory for the entire vector any time that you add an element

Some functions that vectors have are:
  • .pop_back(): pops the last element
  • .push_back(): pushes element into vector
  • .begin(): puts iterator at beginning
  • .erase(): is able to erase elements
  • .empty(): checks if vector is empty
  • .size(): returns the size of vector
