A queue is a basic data structure used in programming. Even in a grocery store you can think of this as a line. The first in the line, like a queue, is the first to be served. To demonstrate how it accesses data, a queue is also called a FIFO( First In First Out).

There are various ways to implement a queue. The basic linked list implementation uses a singly-linked list with a tail pointer to keep track of the back of the queue and works off from there.

A real-world example of a queue may be a one-way road, where the vehicle enters first, leaves first. More real- world examples can be seen at ticket windows and bus stops as queues.

Some functions that queues have are:
  • .pop(): pops the next element
  • .push(): pushes element into queue
  • .front(): accesses front element
  • .back(): accesses back element
  • .empty(): checks if queue is empty
  • .size(): returns the size of queue
